# cumulative-area

当报告需要“累积面积图（ Cumulative Area ）”时使用这个模块。

- 引擎: token-report
- 来源: showroom/report.html §58
- 类型: Chart.js canvas
- Canvas ids: cumArea
- DOM ids: 无
- 脚本: script.js

仅当报告内容需要这种可视化形态时加载本模块。交付前必须替换示例数据、文案、标签、单位和 ARIA 文本。
