报刊式开场段（Newspaper Lede）

最简单也最具仪式感的排版——首字母放大、首行小型大写字母。适合放在报告的开头作为"叙事入口"。

在 2026 年 5 月这个时间点回望，开源编程 Agent 的过去十二个月经历了一场惊人的洗牌。Cursor 与 Windsurf 作为商业 IDE 的代表，一度被认为代表了"AI 编程的未来形态"。但 Anthropic 在 2025 年 2 月发布的 Claude Code 重新定义了这场游戏——它选择了一条没人押注的路：纯命令行、不做 IDE、不做扩展。结果是十二个月内 123k stars、3M 周活，并直接催生了 OpenCode、Crush、Goose、Pi 等一整代"终端原生"开源工具。这份调研记录了这场迁移的核心数据。
