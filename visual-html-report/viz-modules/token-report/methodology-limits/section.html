§ 10 · 调研方法

怎么搞的

数据来源

  • 每一项 stars / forks / 许可证：直接看 GitHub repo 页（2026-05-15）
  • "最新发布"和活跃度：看 Releases 页和 Recent Issues
  • fork 关系：靠 README、Wikipedia、Hacker News 讨论
  • 性能 / SWE-bench：项目官方 README 报告

已知偏差

  • Star 数有刷量噪声（参见 CMU 2026 ICSE 论文）
  • 本表没有覆盖商业闭源工具：Cursor、Windsurf、Devin、Replit Agent
  • 没有覆盖国产小工具：豆包、CodeGeeX 这类
  • Pi-mono 星数较小，搜索结果中数据不一致，取保守估计
